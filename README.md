# Prueba técnica para InSalud

## By David Urbano

## Cómo correr el proyecto

```bash
# Instalar dependencias
npm install

# Correr en modo desarrollo
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) en el navegador.
